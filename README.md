# avito-2025-autumn-intership
Test task for Avito backend intership (autumn 2025).

## Допущения

- В какой ситуации возможна ошибка "команда уже существует", если ручка `GET /team/add` также может обновлять список пользователей? - В случае, если список пользователей не изменился, будет возвращаться ошибка.
- Что будет, если пользователь является ревьювером большого количества pr? - Ручка будет возвращать до `max_pr_in_resp` последних pr, в которых пользователь назначен ревьювером.
- Что если попытаться добавить в команду пользователя, который уже участник другой команды? - Ручка будет возвращать ошибку.
- Что если из команды будет исключен автор или ревьювер pr? - авторство закрепляется на момент создания mr и не может быть изменено. Если пользователь являлся ревьювером, после исключения его из команды он останется ревьювером завершенных pr, но будет удален из списка для открытых.
